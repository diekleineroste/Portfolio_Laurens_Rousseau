  • April 13th: Introducing GSAP for Animations

    Today's update involved implementing the GSAP (GreenSock Animation Platform) library. GSAP allows for smooth and performant animations, which I'll be using to create eye-catching transitions, interactive elements, and better guide users through the content. Stay tuned to see how GSAP animations elevate the user experience!

  • April 15th: Enhancing User Engagement with Interactive Overlays

    Today's focus was on refining the user experience by implementing interactive overlays. These overlays, triggered by hovering over designated website sections, provide additional information and visual interest. This subtle animation technique aims to increase user engagement and encourage exploration of different content areas. To achieve these animations, I utilized the GSAP animation library for its smooth transitions and customizable easing effects. Additionally, I prioritized code efficiency by employing reusable functions and targeted event listeners, ensuring optimal website performance. While seemingly minor, these final touches contribute significantly to a more engaging and enjoyable user experience.
